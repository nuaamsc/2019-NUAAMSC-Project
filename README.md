# MSC寒假成长计划
## 项目简介
###### 1. 由于我们今年的招新计划的推迟至12月初，成员名单确定之后已经到12月中旬了，很抱歉这学期俱乐部没能给大家更好的体验，我们希望推出寒假计划激起大家学习的动力，成为更加优秀的自己。
* 项目进度OpenSource：每个Project都应该在俱乐部的Github中都建立仓库，每次学完一部分内容做一个commit
* 文档撰写：建议使用markdown
* 代码托管平台：github
* 代码版本控制工具：git
###### 2. 希望大家都能参与进来。
